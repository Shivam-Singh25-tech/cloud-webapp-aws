# Cloud Web App on AWS

This project demonstrates a production-style cloud architecture using AWS.

## Architecture

S3 → CloudFront → Load Balancer → EC2 → RDS

## Technologies

- AWS
- Terraform
- Node.js
- MySQL
- GitHub

## Features

- Infrastructure as Code
- Load balancing
- Auto scaling
- Cloud storage
- Managed database

## Deployment

1. Deploy infrastructure with Terraform
2. Deploy backend on EC2
3. Upload frontend to S3
4. Access app through Load Balancer
